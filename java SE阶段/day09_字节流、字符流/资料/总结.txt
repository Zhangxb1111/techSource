能够说出IO流的分类和功能	
	从流向上分类： 输入流和输出流
	从类型上分类： 字节流和字符流。
					字节流读写中文有可能会有问题。
					字符流只能操作文本文件。
能够使用字节输出流写出数据到文件
	1. 创建
	2. 写入
		写一个字节
		写一个字节数组
	3. 释放
能够使用字节输入流读取数据到程序
	1. 创建
	2. 读取
		一次读取一个字节
		fis.read();
		一次读取一个字节数组
		fis.read(bArr);
	3. 释放
能够理解读取数据read(byte[])方法的原理
	byte数组可以看成一个口袋，一次可以装多个字节，所以一次可以读取多个字节。
能够使用字节流完成文件的复制
	一边读一边写。
能够使用FileWirter写数据到文件
	1. 创建
	2. 写入
	3. 刷新
	4. 释放
能够说出FileWriter中关闭和刷新方法的区别
	刷新：仅仅是做的刷新操作。流刷新之后还能使用
	关闭：先刷新再关闭。 流关闭之后不能用
能够使用FileWriter写数据的5个方法
	write(字符串)
	write(字符串的一部分)
	write(字符)
	write(字符数组)
	write(字符数组的一部分)
能够使用FileWriter写数据实现换行和追加写
	追加写，在输出流构造方法的第二个参数传递true。
	换行： \r\n
能够使用FileReader读数据
	1. 创建
	2. 读取
		一次读取一个字符
		fis.read();
		一次读取一个字符数组
		fis.read(cArr);
	3. 释放
能够使用FileReader读数据一次一个字符数组
	和字节流读取一个字节数组一样。
能够使用Properties的load方法加载文件中配置信息
	1. 创建Properties
	2. 创建输入流
	3. 调用load方法传递输入流
	4. 释放资源。
	5. 打印Properties集合中的内容。
	
重点：
	Properties的load方法，getProperties，setProperties
	字节流和字符流的使用
