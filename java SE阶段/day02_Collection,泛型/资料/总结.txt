能够说出集合与数组的区别
	数组： 长度固定， 可以存放任意类型的数据。
	集合： 长度可变， 只能存放引用类型的数据
说出Collection集合的常用功能
	最常用
	add
	size
	
	其他：
	remove
	contains
	clear
	toArray
	isEmpty

能够使用迭代器对集合进行取元素
	1. 调用集合的iterator方法，获取集合对应的迭代器
	2. 调用迭代器的hasNext方法判断是否还有没有元素可以获取
	3. 如果有元素可以获取，那么就调用next方法进行获取，并且把光标向后移动一个位置。
能够使用集合存储自定义类型
	集合存储Student，Person
能够使用foreach（增强for）循环遍历集合
	格式：

能够使用泛型定义集合对象
	 ArrayList<....>
能够理解泛型上下限
	<? extends A> 上限， 只能是A类或者A类的子类
	<? super B> 下限， 只能是B类或者B的父类
能够阐述泛型通配符的作用
	? 可以匹配任意类型的泛型。
